# Black Candle Labs

A landing page for data recovery and remote PC support business.

## Features
- Clean, responsive design
- Header with logo placeholder
- Hello World section
- Services listing
- Contact section
- Mobile-friendly layout

## Setup

1. **Add your logo**: Replace `logo.png` with your business logo (50x50px recommended)
2. **Add your favicon**: Replace `favicon.ico` with your logo as a favicon
3. **Update business details**: Edit `index.html` with your actual business information
4. **Customize colors**: Modify `styles.css` to match your brand

## Deployment with GitHub Pages

1. Go to your repository **Settings**
2. Navigate to **Pages** section
3. Select **Deploy from a branch**
4. Choose `main` branch and `/root` folder
5. Your site will be live at your GitHub Pages URL

## Connect Your Domain

1. In Cloudflare, add your domain records pointing to GitHub Pages
2. In repository Settings → Pages, add your custom domain
3. Enable "Enforce HTTPS"
